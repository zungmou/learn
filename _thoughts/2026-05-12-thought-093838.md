三角形的根号形式

## sin(θ)

| **角度** | **sin(θ) 根号形式** |
|---|---|
| 0° | 0 |
| 15° | $$\frac{\sqrt{6} - \sqrt{2}}{4}$$ |
| 30° | $$\frac{1}{2}$$ |
| 45° | $$\frac{\sqrt{2}}{2}$$ |
| 60° | $$\frac{\sqrt{3}}{2}$$ |
| 75° | $$\frac{\sqrt{6} + \sqrt{2}}{4}$$ |
| 90° | 1 |
| 105° | $$\frac{\sqrt{6} + \sqrt{2}}{4}$$ |
| 120° | $$\frac{\sqrt{3}}{2}$$ |
| 135° | $$\frac{\sqrt{2}}{2}$$ |
| 150° | $$\frac{1}{2}$$ |
| 165° | $$\frac{\sqrt{6} - \sqrt{2}}{4}$$ |
| 180° | 0 |

规律一眼就能看出来：
- **0° → 90° 递增**：0 → (√6−√2)/4 → 1/2 → √2/2 → √3/2 → (√6+√2)/4 → 1
- **90° → 180° 对称递减**：105°=75°、120°=60°、135°=45°、150°=30°、165°=15°、180°=0°

## cos(θ)

| **角度** | **cos(θ) 根号形式** |
|---|---|
| 0° | 1 |
| 15° | $$\frac{\sqrt{6} + \sqrt{2}}{4}$$ |
| 30° | $$\frac{\sqrt{3}}{2}$$ |
| 45° | $$\frac{\sqrt{2}}{2}$$ |
| 60° | $$\frac{1}{2}$$ |
| 75° | $$\frac{\sqrt{6} - \sqrt{2}}{4}$$ |
| 90° | 0 |
| 105° | $$-\frac{\sqrt{6} - \sqrt{2}}{4}$$ |
| 120° | $$-\frac{1}{2}$$ |
| 135° | $$-\frac{\sqrt{2}}{2}$$ |
| 150° | $$-\frac{\sqrt{3}}{2}$$ |
| 165° | $$-\frac{\sqrt{6} + \sqrt{2}}{4}$$ |
| 180° | $$-1$$ |

跟 sin 对照着看有意思：

- **0°→90°**：sin 从 0 爬到 1，cos 从 1 滑到 0，值刚好反过来——sin(θ) = cos(90°−θ)
- **90°→180°**：sin 对称下降回 0，cos 继续降到 −1，变成负的——但根号形式一样，只是符号变了
- **0°→180° 整体**：cos 是 sin 向左平移 90° 的版本，即 cos(θ) = sin(θ + 90°)

## tan(θ)

| **角度** | **tan(θ) 根号形式** |
|---|---|
| 0° | 0 |
| 15° | $$2 - \sqrt{3}$$ |
| 30° | $$\frac{\sqrt{3}}{3}$$ |
| 45° | 1 |
| 60° | $$\sqrt{3}$$ |
| 75° | $$2 + \sqrt{3}$$ |
| 90° | 不存在 |
| 105° | $$-(2 + \sqrt{3})$$ |
| 120° | $$-\sqrt{3}$$ |
| 135° | $$-1$$ |
| 150° | $$-\frac{\sqrt{3}}{3}$$ |
| 165° | $$\sqrt{3} - 2$$ |
| 180° | 0 |

对照着看很有意思：

- **0°→90°**：从 0 单调递增到无穷大，15°→75° 的规律是 $$2-\sqrt{3} \to \frac{\sqrt{3}}{3} \to 1 \to \sqrt{3} \to 2+\sqrt{3}$$
- **90° 是渐近线**：从正无穷直接跳负无穷
- **90°→180°**：从负无穷递增回 0，数值上是前半段加个负号，但顺序反过来——165° 对应 15° 的负值，150° 对应 30° 的负值，以此类推
- **周期对称**：tan(180°−θ) = −tan(θ)，所以 105° = −tan(75°)、120° = −tan(60°)……
