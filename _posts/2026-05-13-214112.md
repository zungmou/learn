### 🪙 资源类

| 秘籍 | 效果 |
|------|------|
| `show me the money` | **+10,000 矿和气**（最经典那个） |
| `whats mine is mine` | 矿产无限 |
| `breathe deep` | 高能瓦斯无限 |
| `food for thought` | 突破补给限制，无限造兵 |

### 👁️ 地图 & 视野

| 秘籍 | 效果 |
|------|------|
| `black sheep wall` | **地图全开** |
| `war aint what it used to be` | 关闭战争迷雾 |

### ⚡ 战斗 & 单位

| 秘籍 | 效果 |
|------|------|
| `power overwhelming` | **无敌**（懂的都懂） |
| `operation CWAL` | 快速建筑/研究 |
| `the gathering` | 法力能量无限 |
| `noglues` | 敌人不能放魔法 |
| `medieval man` | 单位无限生产 |
| `modify the phase variance` | 可造所有建筑 |
| `something for nothing` | 解锁所有可生产选项 |

### 🎯 任务控制

| 秘籍 | 效果 |
|------|------|
| `there is no cow level` | **直接过关** |
| `game over man` | 任务失败 |
| `staying alive` | 任务无法完成 |
| 输入 `ophelia` + Enter，再输关卡名 | **跳关** |
